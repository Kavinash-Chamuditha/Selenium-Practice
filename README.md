# 🚀 Selenium Practice (Java)

This repository contains my hands-on practice with **Selenium WebDriver using Java**.
It focuses on real-world automation scenarios and core Selenium concepts for web testing.

---

## 🧪 Technologies Used

* Java
* Selenium WebDriver
* IntelliJ IDEA
* Maven

---

## 📂 Features Covered

### 🔹 Web Elements & Locators

* ID, Name, XPath, LinkText
* TextBox, Buttons, Dropdowns
* Links & Navigation

### 🔹 Browser Interactions

* Alerts handling
* Frames & iFrames
* Multiple Windows/Tabs
* Radio Buttons & Checkboxes

### 🔹 User Actions

* Mouse Actions
* Keyboard Actions

### 🔹 File Handling

* File Upload
* File Download
* Broken Links & Images Detection

### 🔹 Advanced Topics

* Date Picker (Calendar handling)
* Web Tables (data extraction & validation)
* JavaScriptExecutor (scrolling, actions)

### 🔹 Wait Mechanisms

* Implicit Wait
* Explicit Wait
* Fluent Wait

### 🔹 Screenshot Automation

* Full Page Screenshot
* WebElement Screenshot

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Kavinash-Chamuditha/Selenium-Practice.git
```

2. Open in IntelliJ IDEA

3. Install dependencies:

```
mvn clean install
```

4. Run any test class directly
